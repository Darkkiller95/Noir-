# Dark.killer - WhatsApp Bot (Baileys)

**Dark.killer** est un bot WhatsApp multi-commandes, prototype pour quiz, images, APK, et autres fonctionnalités, déployable localement ou sur Katabump via Docker.

---

## Fonctionnalités principales

- Quiz multi-catégories : animé, animaux, histoire-RDC, géographie Afrique, science, musique, sport, jeu vidéo  
- Gestion d’images : liste, envoi, placeholders pour génération IA  
- Gestion d’APK : liste, envoi via lien  
- Profil utilisateur simple : pseudo  
- Boutique simple (exemple)  
- Menu complet VANSBOT  
- Broadcast réservé aux owners/admins  

---
